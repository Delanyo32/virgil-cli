The query targets TypeScript/JavaScript grammar node kinds (for_statement, for_in_statement, while_statement, do_statement). Languages whose grammars use different node kinds (e.g., Rust's for_expression, Python's for_statement but different nesting) will have the query silently rejected by execute_match_pattern -- accepted limitation per D-06.

Expect higher false positive rate.",

    {
