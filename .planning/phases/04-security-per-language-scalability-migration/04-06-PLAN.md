Create 10 JSON files. All use "languages": ["c"]. Read each Rust source file and translate tree-sitter patterns.

1. **format_string_c.json** -- pipeline: "format_string". Rust detects printf/fprintf/sprintf with non-literal format arg. Match call_expression to printf family. Pattern: "format_string_vulnerability", severity: "error".

2. **c_command_injection_c.json** -- pipeline: "c_command_injection". Rust detects system()/popen() with dynamic arg + sprintf+system pattern. Simplified per D-07: match system()/popen() calls where first arg is not string_literal. Pattern: "command_injection_call", severity: "error". Document: "Simplified: sprintf+system two-phase pattern not expressible in single match_pattern."

3. **c_weak_randomness_c.json** -- pipeline: "c_weak_randomness". Rust detects rand()/random() calls. Pattern: "weak_randomness", severity: "warning".

4. **c_buffer_overflow_security_c.json** -- pipeline: "c_buffer_overflow_security". Rust detects gets()/strcpy()/strcat()/sprintf() calls. Pattern: "buffer_overflow_risk", severity: "error".

5. **c_integer_overflow_c.json** -- pipeline: "c_integer_overflow". Rust detects arithmetic on values from function params without bounds check. Simplified: match binary_expression with arithmetic operators. Pattern: "unchecked_arithmetic", severity: "warning".

6. **c_toctou_c.json** -- pipeline: "c_toctou". Rust detects access()/stat() followed by open(). Simplified: match access()/stat() calls. Pattern: "toctou_check", severity: "warning".

7. **c_memory_mismanagement_c.json** -- pipeline: "c_memory_mismanagement". Rust detects free() without NULL check, double-free patterns. Simplified: match free() calls. Pattern: "memory_mismanagement", severity: "warning".

8. **c_path_traversal_c.json** -- pipeline: "c_path_traversal". Rust detects path concatenation with user input. Simplified: match strcat/sprintf calls. Pattern: "path_traversal_risk", severity: "warning".

9. **c_uninitialized_memory_c.json** -- pipeline: "c_uninitialized_memory". Rust detects malloc() result used without memset/calloc. Simplified: match malloc() calls. Pattern: "uninitialized_memory", severity: "warning".

10. **memory_leak_indicators_c.json** -- pipeline: "memory_leak_indicators", category: "scalability". Simplified per D-09: match malloc/calloc/realloc/strdup/fopen calls broadly. Pattern: "potential_memory_leak", severity: "warning". Document: "Simplified: flags allocation calls without verifying corresponding free/fclose."

Run `cargo test`.
