**Shared JS/TS security pipelines (7 files, each with "languages": ["javascript", "jsx", "typescript", "tsx"]):**

1. **code_injection_javascript.json** -- pipeline: "code_injection", category: "security". Rust detects eval() with non-literal, new Function(), setTimeout with string. Simplified: match call_expression where function name is eval/Function/setTimeout/setInterval. Pattern: "code_injection_call", severity: "error".

2. **command_injection_javascript.json** -- pipeline: "command_injection", category: "security". Rust detects exec/execSync/execFileSync method calls + spawn with shell:true. Simplified: match member_expression method calls to exec/execSync/execFileSync. Pattern: "exec_command_injection", severity: "error".

3. **path_traversal_javascript.json** -- pipeline: "path_traversal", category: "security". Rust detects path.join/path.resolve with dynamic args. Simplified: match member_expression calls to join/resolve. Pattern: "unvalidated_path_operation", severity: "warning".

4. **prototype_pollution_javascript.json** -- pipeline: "prototype_pollution", category: "security". Rust detects for-in with subscript assignment, Object.assign with JSON.parse. Simplified: match subscript_expression inside for_in_statement. Pattern: "prototype_pollution_risk", severity: "warning".

5. **redos_resource_exhaustion_javascript.json** -- pipeline: "redos_resource_exhaustion", category: "security". Rust detects RegExp constructor, exec() on regex. Simplified: match `new_expression` with `RegExp` constructor. Pattern: "dynamic_regex_construction", severity: "warning".

6. **insecure_deserialization_javascript.json** -- pipeline: "insecure_deserialization", category: "security". Rust detects JSON.parse with external source, eval-based deser. Simplified: match call_expression to JSON.parse. Pattern: "insecure_deserialization", severity: "warning".

7. **timing_weak_crypto_javascript.json** -- pipeline: "timing_weak_crypto", category: "security". Rust detects Math.random for crypto, == on auth tokens, MD5/SHA1. Simplified: match member_expression calls to Math.random and calls to createHash. Pattern: "weak_crypto_usage", severity: "warning".

**TypeScript-only security pipelines (2 files, "languages": ["typescript", "tsx"]):**

8. **type_system_bypass_typescript.json** -- pipeline: "type_system_bypass", category: "security". Rust detects @ts-ignore, @ts-nocheck, @ts-expect-error. Read the Rust file for exact patterns. Pattern: "type_system_bypass", severity: "warning".

9. **unsafe_type_assertions_security_typescript.json** -- pipeline: "unsafe_type_assertions_security", category: "security". Rust detects `as unknown as T` double-cast. Read the Rust file for exact pattern. Pattern: "unsafe_type_assertion", severity: "warning".

**Scalability pipelines (2 files):**

10. **memory_leak_indicators_javascript.json** -- pipeline: "memory_leak_indicators", category: "scalability", languages: ["javascript", "jsx"]. Simplified per D-09: match addEventListener and setInterval call_expression nodes unconditionally. Pattern: "potential_memory_leak", severity: "warning". Document: "Simplified from Rust: flags addEventListener/setInterval calls without verifying removeEventListener/clearInterval presence. Higher false positive rate."

11. **memory_leak_indicators_typescript.json** -- pipeline: "memory_leak_indicators", category: "scalability", languages: ["typescript", "tsx"]. Same patterns as JavaScript version.
