**Per-pipeline instructions:**

1. **error_swallowing_go.json** -- pipeline: "error_swallowing". Match blank identifier assignments from function calls: `(assignment_statement left: (expression_list (blank_identifier) @blank))`. Pattern: "swallowed_error", severity: "warning".

2. **god_struct_go.json** -- pipeline: "god_struct". Use select: "symbol", where: {kind: ["struct"]}, exclude: {is_test_file: true}. Then count fields via group_by/count or use match_pattern on struct fields. Pattern: "god_struct", severity: "warning".

3. **goroutine_leak_go.json** -- pipeline: "goroutine_leak". Match go statements: `(go_statement) @go`. Pattern: "goroutine_leak_risk", severity: "warning". Document: simplified; flags all goroutine launches without checking for context cancellation.

4. **init_abuse_go.json** -- pipeline: "init_abuse". Match init function definitions: `(function_declaration name: (identifier) @name)` filtered to name "init". Pattern: "init_function_abuse", severity: "info".

5. **magic_numbers_go.json** -- pipeline: "magic_numbers". Match integer literals: `(int_literal) @lit`. Pattern: "magic_number", severity: "info".

6. **mutex_misuse_go.json** -- pipeline: "mutex_misuse". Match mutex method calls: `(call_expression function: (selector_expression field: (field_identifier) @method))` for Lock/Unlock. Pattern: "mutex_misuse", severity: "warning".

7. **naked_interface_go.json** -- pipeline: "naked_interface". Match empty interface types: `(interface_type) @iface`. Pattern: "naked_interface", severity: "info". Document: flags all interface{} uses.

8. **stringly_typed_config_go.json** -- pipeline: "stringly_typed_config". Match string parameters in functions. Pattern: "stringly_typed_config", severity: "info".

9. **concrete_return_type_go.json** -- pipeline: "concrete_return_type". Match function declarations returning concrete types instead of interfaces. Pattern: "concrete_return_type", severity: "info".

10. **context_not_propagated_go.json** -- pipeline: "context_not_propagated". Match function declarations with context.Context parameter. Pattern: "context_not_propagated", severity: "info".

11. **dead_code_go.json** -- pipeline: "dead_code". Use select: "symbol", where: {exported: true}, exclude: {is_test_file: true}. Pattern: "potentially_dead_export", severity: "info".

12. **duplicate_code_go.json** -- pipeline: "duplicate_code". Use select: "symbol", where: {kind: ["function", "method"]}. Pattern: "potential_duplication", severity: "info". Document: simplified from Rust rolling-hash.

13. **coupling_go.json** -- pipeline: "coupling". Match import declarations: `(import_declaration) @import`. Pattern: "high_coupling", severity: "info".
